VSAS

 http://levibostian.com/VSAS/

Video Surveillance Alert System built for software engineering class.

Motion is detected via webcam which triggers video recording. After motion has stopped, video is compiled and uploaded to Dropbox. Email is then send to admin of system with summary of recorded footage and link to VSAS website where they can view video footage.

VSAS created for CS2720 Software Engineering class Spring 2013.

tech: Python, FFmpeg, OpenCV, Dropbox API, PHP, Software Engineering tech such as SRS documentation, risk assessment, data flow diagrams, etc.
